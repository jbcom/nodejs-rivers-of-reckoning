Quickstart Guide
================

Welcome to **Rivers of Reckoning**! This guide will help you get into the game 
as quickly as possible.

1. Install Dependencies
-----------------------

First, make sure you have the necessary libraries installed:

.. code-block:: bash

   pip install pygame-ce pygbag opensimplex esper

2. Run the Game
---------------

From the root of the repository, run:

.. code-block:: bash

   python main.py

3. Choose Your Adventure
------------------------

On the title screen, press **ENTER** to generate a new procedural world and 
start exploring.

4. Basic Controls
-----------------

* **Arrow Keys**: Move your character.
* **ESC**: Pause the game or return to the menu.
